# Project 2: Trading Signals Using Z-Score

## Description
This is a beginner-friendly Python project that generates *buy and sell trading signals* using the *Z-score* of stock closing prices.  
The project calculates *rolling mean* and *standard deviation, computes the **Z-score, generates **buy/sell signals, and can perform a **simple backtest* to evaluate trading performance.

## Features
- Reads stock price data from a CSV file (or can be extended to use Yahoo Finance API)
- Calculates Z-score for identifying buy/sell signals
- Simple backtesting of trades
- Outputs performance metrics like total return, number of trades, win rate, and average profit per trade
